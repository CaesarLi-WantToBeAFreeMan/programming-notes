1. unique constraint:
      ensures that all values in a column are all different
2. syntax:
      i. create a table:
         CREATE TABLE tableName(columnName variable unique...);
      ii. add constraint:
         ALTER TABLE tableName ADD constraint UNIQUE(columnName);
